# Real-Time Website UI - Complete Redesign

## Design Philosophy
Creating a professional, production-ready IT training institute website inspired by industry leaders like Udemy, Coursera, and modern SaaS platforms.

## Design System

### Colors
- **Primary**: #6366f1 (Indigo)
- **Secondary**: #8b5cf6 (Purple)
- **Accent**: #06b6d4 (Cyan)
- **Success**: #10b981 (Green)
- **Background**: #ffffff (White)
- **Surface**: #f9fafb (Light Gray)
- **Text Primary**: #111827 (Dark Gray)
- **Text Secondary**: #6b7280 (Medium Gray)

### Typography
- **Headings**: 'Inter', sans-serif (700-900 weight)
- **Body**: 'Inter', sans-serif (400-600 weight)
- **Code**: 'Fira Code', monospace

### Components

#### 1. Navigation
- Sticky header with blur background
- Logo on left, menu items center, CTA button right
- Mobile hamburger menu
- Smooth scroll behavior
- Active link highlighting

#### 2. Hero Section
- Full-width with background image/gradient
- Large heading with gradient text
- Subheading and CTA buttons
- Trust indicators (student count, ratings, companies)
- Floating elements/illustrations

#### 3. Feature Cards
- Clean white cards with subtle shadows
- Icon at top with gradient background
- Hover effects with lift and glow
- Consistent spacing and alignment

#### 4. Course Cards
- Image at top with overlay on hover
- Badge for status (Bestseller, New, Trending)
- Course title, description, meta info
- Pricing and CTA button
- Progress bar for enrolled courses

#### 5. Forms
- Clean input fields with focus states
- Floating labels
- Validation messages
- Submit button with loading state
- Success/error notifications

#### 6. Footer
- Multi-column layout
- Company info, quick links, contact
- Social media icons
- Newsletter signup
- Copyright and legal links

## Page-by-Page Design

### Home Page
1. **Hero**: Large heading, subheading, 2 CTAs, trust badges
2. **Features**: 4-6 feature cards in grid
3. **Popular Courses**: 3 course cards with "View All" link
4. **Stats**: Numbers with icons (students, courses, instructors, success rate)
5. **Testimonials**: 3 testimonial cards with photos
6. **Companies**: Logo grid of hiring partners
7. **CTA**: Final call-to-action section

### Courses Page
1. **Header**: Page title, description, filter/search
2. **Filters**: Sidebar with category, level, duration filters
3. **Course Grid**: Responsive grid of course cards
4. **Pagination**: Page numbers at bottom
5. **Enrollment Form**: Modal or separate page

### About Page
1. **Hero**: Company mission and vision
2. **Story**: Timeline or narrative section
3. **Team**: Grid of instructor cards with photos
4. **Values**: Icon cards with company values
5. **Technologies**: Tech stack with icons
6. **Achievements**: Stats and milestones

### Projects Page
1. **Header**: Page title and description
2. **Project Grid**: Cards with images, titles, descriptions
3. **Filters**: By technology, category
4. **Project Details**: Modal or separate page with full info

### Internships Page
1. **Header**: Page title and benefits
2. **Internship Cards**: Available positions
3. **Application Form**: Clean, multi-step form
4. **Requirements**: What we're looking for
5. **Process**: Application process timeline

### Contact Page
1. **Hero**: Contact heading and description
2. **Contact Form**: Name, email, phone, message
3. **Contact Info**: Address, phone, email with icons
4. **Map**: Embedded Google Maps (optional)
5. **Social Links**: Social media buttons
